# Exam, Python elective Spring 2019
Jeg har valgt assignment 1 'Crawl a website and save the content in markdown'

Jeg har udvalgt https://clbokea.github.io/exam/index.html, som den hjemmeside jeg trækker data fra.

Når projektet er klonet, skal linje 41 ændres til den path filerne ønskes gemt, ellers skal der ikke gøres andet end at køre det. Hvis kun exam.py hentes vil den automatisk oprette en .md fil kaldet WebScrape.md, hvor de forskellige ting fra hjemmesiden gemmes i. Hvis alle filer fra repositoriet hentes, vil den overskrive det der er i WebScape.md.
